# 2026-05-27

## URL Token 免密登录

新增 URL token 免密登录功能，允许通过带 token 的链接直接登录系统：

**后端改动** (`agent_core/main.py`):
- 新增 `LOGIN_TOKEN_EXPIRY_SECONDS = 300` 常量（5 分钟有效期）
- 新增 `LOGIN_TOKEN_ERROR_HTML` 错误提示页面
- 新增 `GET /auth/token-login` 端点：验证 URL query 参数 `token`（HMAC-SHA256），设置 session cookie，302 重定向到主页
- 将 `/auth/token-login` 加入 `_auth_exempt_path` 放行列表
- 修复：在重定向 Response 对象上调用 `set_cookie()` 而非参数 `response`（原写法 cookie 不生效）

**CLI 工具** (`generate_login_url.py`):
- 读取 `~/.desktop_agent/auth.json` 中的 secret，生成短时效 token
- 支持 `--host`、`--port`、`--expires`、`--qr`、`--copy` 参数
- 输出可直接在浏览器打开的链接

**验证结果**:
- 有效 token → 302 + `Set-Cookie: desktop_agent_session=...` ✓
- 无效 token → 错误页面 401 ✓
- 缺 token → 错误页面 400 ✓
