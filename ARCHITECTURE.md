## 四、Agent 执行流程 (ReAct Loop)

```
用户发送消息
    │
    ▼
┌──────────────────────────────────────────────────────┐
│  POST /run/stream                                    │
│  1. 创建/获取会话                                     │
│  2. 保存用户消息到 session_store                      │
│  3. switch_thread(session_id)                        │
│  4. 调用 agent.stream_run(message)                   │
└───────────────────────┬──────────────────────────────┘
                        │
                        ▼
┌──────────────────────────────────────────────────────┐
│              LangGraph ReAct Agent                   │
│                                                      │
│  ┌────────────────────────────────────────────────┐  │
│  │  System Prompt = 基础提示词 + 技能描述块         │  │
│  └────────────────────────────────────────────────┘  │
│                        │                             │
│                        ▼                             │
│  ┌────────────────────────────────────────────────┐  │
│  │           LLM 推理 (ChatOpenAI)                │  │
│  │                                                │  │
│  │  输入: System Prompt + 历史消息 + 用户消息      │  │
│  │  输出: 文本回复  或  工具调用指令                │  │
│  └────────────┬───────────────┬───────────────────┘  │
│               │               │                      │
│          (纯文本回复)     (工具调用)                   │
│               │               │                      │
│               ▼               ▼                      │
│          返回最终回复    ┌─────────────────┐          │
│                         │   工具执行层     │          │
│                         │                 │          │
│                         │  ┌───────────┐  │          │
│                         │  │file_tools │  │          │
│                         │  │code_tools │  │          │
│                         │  │sys_tools  │  │          │
│                         │  └─────┬─────┘  │          │
│                         │        │        │          │
│                         │        ▼        │          │
│                         │   工具返回结果   │          │
│                         └────────┬────────┘          │
│                                  │                   │
│                                  ▼                   │
│                    将结果注入消息历史                   │
│                    继续 LLM 推理 ◀───────────┘        │
│                                                      │
│  循环直到 LLM 输出纯文本（无工具调用）                  │
│                                                      │
└───────────────────────┬──────────────────────────────┘
                        │
                        ▼
┌──────────────────────────────────────────────────────┐
│  SSE 流式事件                                         │
│                                                      │
│  → tool_start   { tool, args, thought, step }        │
│  → tool_result  { tool, result }                     │
│  → token        { content }     (逐 token 输出)      │
│  → done         { content }     (最终完整回复)        │
│  → [DONE]                    (流结束标记)             │
│                                                      │
└───────────────────────┬──────────────────────────────┘
                        │
                        ▼
┌──────────────────────────────────────────────────────┐
│  前端渲染                                             │
│                                                      │
│  1. tool_start  → 显示工具调用卡片（折叠式）           │
│  2. tool_result → 更新卡片状态为 ✅ 完成              │
│  3. token       → 实时追加 Markdown 渲染              │
│  4. done        → 最终确认渲染                        │
│  5. 刷新侧边栏统计（Token/费用/会话列表）              │
│                                                      │
└──────────────────────────────────────────────────────┘
```

## 十二、SSE 事件协议

```
┌──────────────────────────────────────────────────────────────────┐
│  流式事件格式 (Server-Sent Events)                               │
│                                                                  │
│  每个事件格式: data: {JSON}\n\n                                   │
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐ │
│  │ event: tool_start                                           │ │
│  │ data: {                                                     │ │
│  │   "type": "tool_start",                                     │ │
│  │   "tool": "read_file",        ← 工具名称                    │ │
│  │   "args": {"path": "xxx"},    ← 调用参数                    │ │
│  │   "thought": "...",           ← AI 思考过程 (可选)          │ │
│  │   "step": 1                   ← 步骤序号                    │ │
│  │ }                                                           │ │
│  ├─────────────────────────────────────────────────────────────┤ │
│  │ event: tool_result                                          │ │
│  │ data: {                                                     │ │
│  │   "type": "tool_result",                                    │ │
│  │   "tool": "read_file",        ← 工具名称                    │ │
│  │   "result": "文件内容..."     ← 工具返回 (截断至 400 字符)  │ │
│  │ }                                                           │ │
│  ├─────────────────────────────────────────────────────────────┤ │
│  │ event: token                                                │ │
│  │ data: {                                                     │ │
│  │   "type": "token",                                          │ │
│  │   "content": "你好"           ← 单个 token 内容             │ │
│  │ }                                                           │ │
│  ├─────────────────────────────────────────────────────────────┤ │
│  │ event: error                                                │ │
│  │ data: {                                                     │ │
│  │   "type": "error",                                          │ │
│  │   "content": "错误信息"       ← 错误描述                    │ │
│  │ }                                                           │ │
│  ├─────────────────────────────────────────────────────────────┤ │
│  │ event: done                                                 │ │
│  │ data: {                                                     │ │
│  │   "type": "done",                                           │ │
│  │   "content": "完整回复内容"   ← 最终完整回复                │ │
│  │ }                                                           │ │
│  ├─────────────────────────────────────────────────────────────┤ │
│  │ data: [DONE]                  ← 流结束标记                  │ │
│  └─────────────────────────────────────────────────────────────┘ │
│                                                                  │
│  时序: tool_start → (tool_result) → ... → token* → done → [DONE]│
└──────────────────────────────────────────────────────────────────┘
```
