# 推荐一个可在内网部署的桌面 AI 智能体：AgentSmith

> 一个能读写文件、运行代码、搜索网页、管理 Git、接入微信的 AI 助手，全部跑在你的电脑上，数据不出门。

---

## 为什么需要私有部署的 AI 智能体？

过去一年，我用过不少 AI 聊天产品。Claude、ChatGPT、DeepSeek 都很好用——直到我开始让 AI 真正"做事"而不是"聊天"。

我需要一个 AI 能：

- **直接读写我电脑上的文件**，而不是让我复制粘贴
- **运行代码并看到输出**，帮我调试问题
- **搜索网页并抓取正文**，整理信息
- **管理我的 Git 仓库**，帮我提交代码
- **在微信上也能用**，碎片时间也能继续工作

更重要的是——**所有数据都在我的机器上**，不用上传到任何第三方服务器。

这就是我要推荐 **AgentSmith**（项目名）的原因。

---

## AgentSmith 是什么

AgentSmith 是一个**本地/私网部署的桌面 AI 智能体**。后端是 Python（FastAPI + LangGraph），前端是一个简洁的 Web 页面，可以用浏览器直接访问。

它本质上是一个**增强版的 AI 助手**——它在传统对话能力之上，内置了大量工具，让 AI 真正能操作你的电脑环境。

---

## 核心亮点

### 1. 🛠 丰富的工具生态

AgentSmith 内置了十几个工具，覆盖日常开发最常用的场景：

**文件操作**：读写、追加、编辑、搜索、列出文件。大文件只返回摘要和路径，不会撑爆上下文。

**代码执行**：在隔离的 Python 子进程中运行代码并获取输出，支持**实时流式输出**——长时间运行的脚本可以看到中间结果。

**Shell 命令**：直接在你的电脑上执行 shell 命令，自动适配 Linux/macOS（bash）和 Windows（powershell）。内置安全拦截，禁止提权和危险操作。

**网页能力**：搜索互联网（Bing → 搜狗 → DuckDuckGo 自动降级），抓取网页正文。

**Git 管理**：查看状态、diff、log、add、commit、push、revert、merge、checkout——但高风险操作默认不开放。

**子代理**：可以委派专门的 `coder` / `reviewer` / `debugger` / `searcher` 子代理并行处理独立任务，支持**实时日志流**。

**数据库交互**：直接通过自然语言查询 SQLite、PostgreSQL、MySQL，支持列级和行级权限控制。

每个工具的执行过程都在前端以**步骤卡片**的形式展示——耗时、状态、参数、结果一目了然，不再是黑盒。

### 2. 🔒 真正的私有部署

所有数据都在你自己的机器上：

- **模型调用**：对接 OpenAI 兼容接口（可以是本地运行的模型、内网的 LLM 网关、或者云厂商的 API）
- **会话数据**：存储在本地 SQLite
- **工作区文件**：就在你的磁盘上
- **用户认证**：本地配置文件，支持多用户

没有云同步、没有数据上传、没有第三方依赖。

### 3. 👥 多用户支持

一个 AgentSmith 服务可以给团队多人使用：

- 每个用户**独立的工作区、会话记录、用量统计和长期记忆**
- 管理员可以管理用户和配置
- 每个用户还可以**绑定自己的微信 Bot**，token 按用户隔离

### 4. 💬 微信集成

这是我觉得最酷的功能之一——AgentSmith 通过**腾讯官方 iLink 协议**接入微信个人号：

- 扫码一次登录，Token 持久化，**服务重启后自动恢复**
- 不需要公网 IP、不需要 ngrok、不需要服务器——客户端主动长轮询
- 在微信里发消息，AI 自动处理并回复，**支持 `/new`、`/list`、`/switch` 等指令管理会话**
- **多用户支持**：每个平台用户可绑定自己独立的微信 Bot

这意味着你在微信上也能继续工作——发个消息让 AI 查资料、写代码、改文件，和 Web 端完全一样。

### 5. 🧩 Skills 技能扩展系统

AgentSmith 支持通过 `SKILL.md` 文件扩展能力：

- 内置 15+ 个技能（头脑风暴、TDD、代码审查、计划执行等）
- 兼容 oh-my-openagent 和 Superpowers 风格的技能格式
- 侧边栏显示已加载的技能，点击可查看详情
- 加载后自动注入系统提示，AI 知道什么时候该用什么技能

### 6. 🤔 长期记忆

AI 会记住你偏好的回答风格、常用环境信息、项目事实：

- 按用户隔离存储
- **超过 10 天自动清理**，不会无限膨胀
- 也可以在管理面板中手动管理

### 7. 📁 会话独立工作目录

每个会话可以设置自己的工作目录：

- 切换会话时自动恢复对应的工作目录
- 工作目录内的文件操作完全开放
- 工作区外文件需显式授权（点击「授权写入」即可）

## 快速上手

```bash
git clone https://gitee.com/mingwuda/desktop-agent.git
cd desktop-agent
pip install -r requirements.txt

# 配置 API Key
export LLM_API_KEY=sk-your-key
export LLM_MODEL=gpt-4o

# 启动
cd agent_core
python main.py
```

打开 `http://127.0.0.1:8899` 就能看到界面。

也可以一键 Docker 部署：

```bash
cp .env.example .env
# 编辑 .env 填入 API Key
docker compose up -d --build
```
